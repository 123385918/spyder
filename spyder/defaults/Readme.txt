# What is the purpose of this directory?

The configuration file located on this directory allows to define the
system/site defaults for Spyder.

if a file `spyder.ini` is found, it will be loaded as the defaults.

## How to create the file?

  1. Open Spyder
  2. Change configuration preferences and apply them
  3. Go to the Spyder configuration folder `~/.spyder/config/`
  4. Copy the `spyder.ini` file into this directory
